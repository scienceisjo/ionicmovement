⚛ 이온(Ion)이란?

이온(Ion)은 원자나 분자가 전자를 잃거나 얻어서 전하를 띠게 된 입자입니다.


• 원자: 양성자 수 = 전자 수 → 전기적으로 중성

• 전자를 잃으면 → 양성자 수 > 전자 수 → 양이온(Cation, +)

• 전자를 얻으면 → 양성자 수 < 전자 수 → 음이온(Anion, -)


이온이 되어도 원자핵(양성자·중성자)은 변하지 않고, 전자의 수만 변합니다.


이온 화학식: 원소 기호 오른쪽 위에 전하 표시   Na⁺, Mg²⁺, Cl⁻, O²⁻

⚡ 양이온 형성

원자가 전자를 잃으면 양이온이 됩니다.

주로 금속 원소가 양이온을 형성합니다.


• 1족 (H, Li, Na, K): 전자 1개 잃음 → +1

• 2족 (Be, Mg, Ca): 전자 2개 잃음 → +2

• 13족 (Al): 전자 3개 잃음 → +3


전자를 잃으면 직전 비활성 기체와 같은 전자 배치가 됩니다.


Na → Na⁺ + e⁻

나트륨(2-8-1) → 나트륨 이온(2-8)

💗 음이온 형성

원자가 전자를 얻으면 음이온이 됩니다.

주로 비금속 원소가 음이온을 형성합니다.


• 17족 (F, Cl): 전자 1개 얻음 → -1

• 16족 (O, S): 전자 2개 얻음 → -2

• 15족 (N, P): 전자 3개 얻음 → -3


전자를 얻으면 다음 비활성 기체와 같은 전자 배치가 됩니다.


Cl + e⁻ → Cl⁻

염소(2-8-7) → 염화 이온(2-8-8)

💡 핵심 정리

• 이온이 되어도 원자핵은 변하지 않습니다 (양성자·중성자 수 동일)

• 오직 전자의 수만 변합니다

• 전자를 잃으면 (+) 전하, 전자를 얻으면 (-) 전하를 띱니다

• 전하량 = 잃거나 얻은 전자 수

• 화학식: 원소 기호 오른쪽 위에 전하 표시  Na⁺, Cl⁻, Mg²⁺, O²⁻

📌 '전자 수 변화' 읽는 법

• (−) 는 전자를 잃었다는 것을 의미합니다.  예) −1개 = 전자 1개를 잃음 → 양이온

• (+) 는 전자를 얻었다는 것을 의미합니다.  예) +2개 = 전자 2개를 얻음 → 음이온

• 잃거나 얻은 전자의 수 = 이온의 전하량 (이온식에 표시되는 숫자)
